## Getting Started
In `config/config.js` file paste the following variable with their respective values.
```
API_URL=
ACCESS_KEY=
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
